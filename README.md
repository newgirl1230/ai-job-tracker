# AI Job Tracker

AI Job Tracker is an AI-powered job application tracker that helps job seekers evaluate job opportunities and manage their application pipeline.

## Problem

Job seekers often save many job opportunities across different job boards, but it is hard to decide which ones are worth applying to. Job descriptions are long, application status is scattered across emails, spreadsheets, and job boards, and tools like Excel or Notion can track jobs but cannot help prioritize opportunities.

## Core User Flow

1. User creates resume profile
2. User pastes job description
3. AI analyzes resume-JD fit
4. User sees match score and suggestions
5. User saves job opportunity
6. User tracks status through the pipeline

## MVP Features

- Resume Profile
- JD Analyzer
- AI Match Score
- Job Tracker
- Job Detail Page
- Notes and Follow-up
- CSV Export

## Out of Scope for MVP

- Automatic job board scraping
- Auto-apply
- Full email sync
